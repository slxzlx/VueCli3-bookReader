# bookReader
#此项目已经实现了epub格式图书的阅读，点击屏幕左右翻页，跳转进度，设置字体，主题，全文搜索，添加书签，目录浏览等功能。前台书架界面目前实现了图书展示，书架管理，上拉刷新等。使用了better-scroll vantUi等插件
